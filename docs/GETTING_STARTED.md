# Getting Started with AccessibilityBuilder Bot 🚀\n\nWelcome! This guide will help you get started, whether you're on a phone, laptop, or just want to contribute ideas.\n\n## For Phone Users: GitHub Codespaces (Easiest!)\n\n### Step 1: Open GitHub Codespaces\n1. Go to: https://github.com/mnsims1024-lab/mnsims-lab-dream-big-happy\n2. Click **Code** button (green button at top right)\n3. Click **Codespaces** tab\n4. Click **Create codespace on main**\n5. Wait 2-3 minutes for environment to load\n\n### Step 2: You now have a full development environment!\n\nIn the terminal:\n\n```bash\n# Install dependencies\npip install -r backend/requirements.txt\ncd frontend\nnpm install\ncd ..\n\n# Start backend API\npython backend/main.py\n# You'll see: \"Uvicorn running on http://127.0.0.1:8000\"\n\n# In a new terminal, start frontend\ncd frontend\nnpm run dev\n# You'll see a local URL like http://localhost:5173\n```\n\n### Step 3: Click the URL in terminal\nCodespaces automatically creates a forwarded URL - click it to see your app running!\n\n---\n\n## For PC/Mac Users\n\n### Prerequisites\n- Python 3.9+\n- Node.js 16+\n- Git\n\n### Installation\n\n```bash\n# Clone repository\ngit clone https://github.com/mnsims1024-lab/mnsims-lab-dream-big-happy.git\ncd mnsims-lab-dream-big-happy\n\n# Backend setup\ncd backend\npython -m venv venv\n\n# Activate virtual environment\n# On Windows:\nvenv\\Scripts\\activate\n# On Mac/Linux:\nsource venv/bin/activate\n\n# Install dependencies\npip install -r requirements.txt\n\n# Run backend\npython main.py\n# Backend running at http://localhost:8000\n\n# Frontend setup (new terminal)\ncd frontend\nnpm install\nnpm run dev\n# Frontend running at http://localhost:5173\n```\n\n### Access the Application\n\n- **Frontend:** http://localhost:5173\n- **API Docs:** http://localhost:8000/api/docs\n- **OpenAPI:** http://localhost:8000/api/openapi.json\n\n---\n\n## Using the Builder API\n\n### Via API (Programmatic)\n\n```python\nfrom app.core import Builder, BuilderConfig\n\n# Create builder\nconfig = BuilderConfig(wcag_level=\"AA\", voice_enabled=True)\nbuilder = Builder(config)\n\n# Generate app from description\ndescription = \"\"\"\nCreate a medication reminder app for elderly users.\nInclude:\n- Large, easy-to-read font\n- Simple voice commands (say 'remind me')\n- High contrast display\n- Clear audio alerts\n\"\"\"\n\nresult = builder.generate_from_text(description)\n\nif result.success:\n    print(f\"✅ Build successful!\")\n    print(f\"Accessibility score: {result.accessibility_score}%\")\n    print(f\"Components: {result.components}\")\n    print(f\"Generated files: {list(result.generated_code.keys())}\")\nelse:\n    print(f\"❌ Build failed\")\n    for error in result.errors:\n        print(f\"  - {error}\")\n```\n\n### Via HTTP API\n\n```bash\ncurl -X POST \"http://localhost:8000/api/build\" \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\n    \"description\": \"Create a medication reminder app for elderly users\",\n    \"wcag_level\": \"AA\",\n    \"voice_enabled\": true,\n    \"include_tts\": true\n  }'\n```\n\n### Check API Documentation\n\n1. Start the backend: `python backend/main.py`\n2. Open browser: http://localhost:8000/api/docs\n3. You'll see interactive API docs where you can test endpoints!\n\n---\n\n## Validating Code for Accessibility\n\n```python\nfrom app.core import AccessibilityValidator\n\nvalidator = AccessibilityValidator(wcag_level=\"AA\")\n\n# Check HTML\nhtml = '<button>Click me</button>'\nis_valid, issues = validator.validate_html(html)\n\nif not is_valid:\n    print(\"Accessibility issues found:\")\n    for issue in issues:\n        print(f\"  - {issue.rule}: {issue.description}\")\n        print(f\"    Fix: {issue.suggestion}\")\n\n# Get detailed report\nreport = validator.generate_report()\nprint(report)\n```\n\n---\n\n## Auto-Patching Code\n\n```python\nfrom app.core import AutoPatcher\n\npatcher = AutoPatcher()\n\n# Patch HTML\nhtml = '<div onclick=\"alert(1)\">Click me</div>'\npatched_html, patches = patcher.patch_html(html)\n\nprint(f\"Patches applied: {patches}\")\nprint(f\"Patched HTML: {patched_html}\")\n```\n\n---\n\n## Project Structure\n\n```\nmnsims-lab-dream-big-happy/\n├── backend/\n│   ├── app/\n│   │   ├── core/\n│   │   │   ├── builder.py        #
